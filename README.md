# Signal-Desktop-chat-export-manager
The Signal Desktop app usually exports jsonl(JSON lines) files which are quie difficult to read and manage, many online tools availabe for reaiding jsonl or json files do not help, so this is a solution for that.

HOW TO RUN:
copy and paste to the folder containing the main.jsonl file
run: 

<i>python3 signal_chat_viewer.py</i>
